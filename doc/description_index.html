Overview

Emperor’s modern user interface, allows you to customize the appearance of your plot by controlling things like: opacity, sphere scaling, coloring, dimensions being presented among many other.

This document tries to go through an overview of the controls that can be found in the graphical user interface as well as some of the common use cases that Emperor allows you to work with.

The graphical user interface is mainly composed by two areas, sample counter (a) the visualization canvas (b) and the tools and settings area (c), see Figure 1. In the visualization canvas, you’ll find the current plot that you are interacting, by default presenting the first three dimensions of the input data. Modifying the appearance of this plot, can be done using any of the tabs contained in (c).

Figure 1. Emperor’s graphical user interface. (a) sample counter; shows the total number of samples in the dataset and the number of samples that are currently visible. (b) canvas; main plot scene where the data is displayed. © tools; controls that allow you to modify and interact with your data.

Description

Emperor presents multiple tabs on the right sidebar, each of these tabs perform different tasks over the data presented in the visualization canvas.

Key

Displays a list of the sample identifiers that are contained in the current dataset and a box with the color that the sample is currently colored by on screen. Double-clicking this square, will make an indicator appear on screen.

Color

Shows a list of the categories listed under the mapping file column selected in the menu of this tab (for this example DOB). Each category is presented with a box that represent the current color the samples are presented in the main plot. Double clicking in this box, will present a color palette that will allow you to change the colors for the samples.

Vsibility

Shows a list of the categories listed under the mapping file column selected in the menu of this tab, for each of these categories a checkbox and a slider is presented. The checkbox will allow you to remove (unchecked) or add (checked) the samples to the plot, by default all categories are checked. The slider controls the opacity of the samples defined under this category.

Scaling

Shows a list of the categories listed under the mapping file column selected in the menu of this tab, for each of these categories a slider is presented. The slider allows you to control the overall radius of the samples. The values range from 0.2 to 4, when the slider is located at a value of 2, the spheres will be twice as bigger as in the beginning. By default all sliders are set to a value of 1.

Labels

This tab lets you control the appearance of the labels that can be added to each of the samples on-screen. The opacity slider controls the overall transparency that samples will be colored by. The checkbox at the top of this tab will add or remove the labels from the plot.

Axes

Although Emperor is only capable of displaying three dimensions at a time, the dimensions that are being displayed, can be selected from this menu. To update the plot to reflect the latest changes, you’ll have to click the refresh button.

Options

The different options that will affect the overall plot as it is are listed under this tab. Scaling the coordinates by the percent explained, will re-scale the data by multiplying all the values in that dimension by the ratio explained. By default Emperor will try to color samples along a continuous color gradient, to turn this option off, select “Use discrete colors”. The sphere opacity and sphere scale controls override the options in Scaling and Visibility tabs. The last control under this list is the background color of the plot, similarly to changing the color the samples clicking in the box will present a color palette to select a new color.
